Welcome to the DocuSign C# examples using multiple OAuth flows (JWT and Authorization Code Grant).

This launcher demonstrates use of the JWT and Authorization Code Grant OAuth flows and includes multiple usage examples for the DocuSign Admin API.

Admin examples

1. Create new user

This example shows how to create a new user with active status

API method used: Users:createUser

2. How to bulk-export user data

This example shows how to bulk-export user data

API method used: UserExport:getUserListExports

3. Check request status

Check the status of your pending request and, if complete, obtain a URL that you can call to download the CSV file containing the export data.

API method used: UserExport:getUserListExport

4. How to add users via bulk import

How to bulk-import users and add user data from an example CSV (comma-separated value) file into a DocuSign Admin organization.

5. Check the request status

Check the status of your pending request and, if complete, obtain a URL that you can call to download the CSV file containing the data to verify that the updates were made

API method used: UserImport :getBulkUserImportRequest
